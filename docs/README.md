# Documentación Técnica - Solace Client Archetype

Esta carpeta contiene la documentación técnica detallada del proyecto Solace Client Archetype.

## Índice de Documentación

- [**Arquitectura de la Solución**](architecture.md) - Descripción detallada de la arquitectura del sistema

## Propósito

Esta documentación proporciona una visión técnica completa del microservicio Spring Boot que integra con el broker de mensajería Solace, incluyendo:

- Patrones de arquitectura implementados
- Flujos de datos y mensajería
- Diagramas de secuencia detallados
- Componentes y sus interacciones
- Casos de uso de la demo

## Convenciones

- Los diagramas están implementados usando **Mermaid** para facilitar su mantenimiento y visualización
- La documentación sigue el formato **Markdown** estándar
- Los ejemplos de código incluyen comentarios explicativos en español
